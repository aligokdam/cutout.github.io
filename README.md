# Cutout Effect Studio

Transform photographs into bold, flat-color artwork directly in your browser.

Cutout Effect Studio is an open-source creative tool that transforms photographs into simplified, poster-style illustrations inspired by paper collage and screen-print aesthetics.

All processing happens locally in your browser. Your images are never uploaded or stored.

🌐 **Live Demo**  
https://aligokdam.github.io/cutout.github.io/

---

## Features

- 📖 Interactive book-style introduction
- 🎨 Poster-style image transformation
- 🎚️ Adjustable color levels
- ✂️ Edge simplification controls
- ⚡ Real-time preview
- 📂 Drag & Drop image upload
- 💾 PNG & JPG export
- 🔒 Fully client-side processing

---

## How It Works

1. Upload an image.
2. Adjust the processing controls.
3. Preview the result instantly.
4. Export your artwork as PNG or JPG.
